🌍 Global GDP Insights — ML & Data Analysis Project
🧠 Overview

This project analyzes global GDP data to uncover economic patterns and forecast future trends using machine learning models.

The dataset contains GDP (current USD) for countries and regions over time. Since it is nominal GDP, results may reflect both real growth and inflation.

📊 Key Analysis
🔍 Data Exploration
Identified top economies by GDP (e.g., United States, China)
Measured:
Growth rates 📈
Volatility ⚠️
Found strong global upward trends with occasional shocks (e.g., crises)
🌐 Economic Insights
Largest economies dominate global output
Fast-growing countries show strong long-term expansion
Volatility varies by region and economic structure
🤖 Machine Learning Models
Models Used:
Linear Regression
Log-Linear Regression
ARIMA
Prophet
LSTM
🏆 Final Results
Best Model: ✅ Prophet
Close Performers: ARIMA, LSTM
Worst Model: ❌ Linear Regression
📌 Key Takeaways:
Prophet handled trend changes and non-linearity best
ARIMA and LSTM performed well but slightly below Prophet
Simple Linear Regression failed to capture complex GDP growth
🔁 Correlation Analysis
Strong correlations between:
European countries (EU economies move together)
USA & Canada (trade relationship)
Global events (e.g., COVID-19, financial crises) affect most countries simultaneously
📈 Forecasting
Built 5-year forecasts for:
United States
China
India
Prophet produced the most reliable and stable predictions
💡 Final Conclusion
Prophet is the best model for GDP forecasting in this project
Simpler models (like Linear Regression) are not suitable for complex economic time series
Economic data benefits from models that capture:
Trends
Structural changes
Non-linear patterns
⚠️ Limitations
GDP is nominal (not inflation-adjusted)
Limited features (only GDP used)
External factors (policy, wars, inflation) not directly modeled
